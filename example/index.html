📖 使用说明

1. 启动后端服务器：pnpm run example:server

2. 在上方输入框修改 SSE 服务器地址（默认：http://localhost:3001/sse）

3. 点击 "🔌 连接" 按钮，客户端会连接到 SSE 服务器

4. 服务器会每秒发送一条消息，包含不同类型（message、notification、system）

5. 您可以在控制台或页面上查看接收到的消息

6. 支持自动重连机制，网络中断后会自动尝试重新连接

7. 统计栏实时显示连接状态、重连次数和收到的消息数量

8. 示例演示了 once() 方法，首次连接时会显示特殊消息（仅一次）
